# User flows

## 1. Anonymous

- Lands on the site (home or course list).
- Sees list of available courses (title, description, maybe thumbnail).
- **Does not see the course viewer with progress** — that experience is for authorized users only. Either: sees only the course list and must log in / register to open a course, or sees a limited course view (e.g. first block or intro only) with a prompt to log in to continue and save progress.
- Cannot see progress bar, completed blocks, or saved position.
- Can take the course quiz and see result only if quiz is exposed without the full viewer; result is not saved. (If course content is gated, quiz is typically after login.)
- Can open “Register” or “Log in” at any time.

## 2. With account

- Logs in (or registers then logs in).
- **Course viewer with progress is visible only to authorized users.** Sees course list, opens a course, and gets the full viewer: ordered blocks (video, rich text, file) with prev/next, progress bar, completed blocks, saved position.
- Progress is saved; can leave and resume later.
- Can take the course quiz **once**; answers are stored; quiz cannot be retaken.
- **Everyone who completed the course can receive the certificate.** Including quiz results on the certificate is optional. On course completion: can download certificate (PDF).

## 3. Admin

Create/edit courses (and their blocks), quizzes; see enrollments/completions if needed.
